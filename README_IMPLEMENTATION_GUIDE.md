## Key Concepts & Terminology

### The Three Stages

**Stage 1: Manual Operations**
- You manually place each order
- 232 minutes per 11 orders (21 min/order)
- 8-12% error rate
- Antifragile Coefficient: -0.4 (degrades under stress)

**Stage 2: Reactive Automation**
- System automates routine tasks
- 8 minutes per 11 orders (0.73 min/order)
- 2-3% error rate
- Antifragile Coefficient: 0.0 (robust, unchanged by stress)
- 96.6% time savings

**Stage 3: Sovereign Intelligence**
- Multi-agent system predicts and prevents problems
- 0-2 minutes per 11 orders (0.09-0.18 min/order)
- 0.3-0.8% error rate
- Antifragile Coefficient: +0.31 (improves 31% from stress)
- 99.1-100% automation

### Key Metrics

**Antifragile Coefficient (AF)**:
```
AF = (Post-Stress_Performance - Pre-Stress_Performance) / Stress_Magnitude

AF > 0 → Antifragile (gains from stress)
AF = 0 → Robust (unchanged)
AF < 0 → Fragile (weakened)
```

**Second-Order Coherence (SOC)**:
```
SOC = (Meta-Observational_Accuracy × Bias_Correction_Rate) / Observational_Rigidity

SOC > 2.0 → Self-correcting
SOC = 1.0-2.0 → Functional
SOC < 1.0 → Rigid
```

**Prediction Accuracy (PA)**:
```
PA = (True Positives + True Negatives) / Total Predictions

PA > 0.80 → Excellent
PA = 0.60-0.80 → Good
PA < 0.60 → Poor
```

**Stigmergic Efficiency (SE)**:
```
SE = (Coordination_Capacity × Agent_Autonomy) / Communication_Overhead

SE > 10.0 → Excellent coordination scalability
```

**Constructal Number (C)**:
```
C = Actual_Flow_Efficiency / Theoretical_Maximum_Efficiency

C > 0.80 → Near-optimal topology
C = 0.50-0.80 → Functional but improvable
C < 0.50 → Significant optimization opportunities
```

### Common Questions

**Q: Should I build Stage 2 or Stage 3?**
A: See decision tree in SPLANTS_GUIDE_SOVEREIGN_INTELLIGENCE.md (Introduction section). Generally: <100 orders/month → Stage 2, >100 orders/month → Stage 3.

**Q: Can I skip Stage 2 and go directly to Stage 3?**
A: Not recommended. Stage 3 builds on Stage 2 foundation. You'll need the reactive automation layer before adding predictive intelligence.

**Q: How much does Stage 3 really cost per month?**
A: At 100 orders/month: $46-169/month total ($31-51 for Stage 2 base + $15-118 for LLM/agents). At 500 orders/month: $75-245/month.

**Q: Is CrewAI the only agent framework option?**
A: No. Alternatives include LangGraph, AutoGen, and custom implementations. CrewAI is recommended for ease of use and community support.

**Q: Can I use local LLMs instead of OpenAI/Anthropic?**
A: Yes, using Ollama. Trade-offs: $0 cost but lower quality (6/10 vs 9/10) and setup complexity. See OPTIMAL_TOOL_STACK_GUIDE.md Section 5.

**Q: What if I'm not comfortable with Python?**
A: Stage 2 only requires JavaScript (Make.com). Stage 3 requires Python for agent frameworks. Consider learning Python first or hiring a Python developer.
